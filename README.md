# 🧠 Coarsening fMRI Graph to Predict Phenotype Features

## 📌 Overview

This project focuses on analyzing **fMRI (Functional Magnetic Resonance Imaging)** data using graph-based techniques.
The main objective is to **coarsen brain connectivity graphs** and use them to predict **phenotype features** such as cognitive or behavioral traits.

---

## 🚀 Motivation

Brain networks are highly complex and high-dimensional.
Graph coarsening helps in:

* Reducing computational complexity
* Preserving important structural information
* Improving model efficiency for prediction tasks

---

## 🛠️ Tech Stack

* Python
* NumPy

## ⚙️ Workflow

1. Load fMRI dataset
2. Construct brain connectivity graph
3. Apply graph coarsening techniques
4. Extract meaningful features
5. Train ML models for phenotype prediction
6. Evaluate performance

---

## 📊 Results

* Reduced graph size while preserving key structures
* Improved computational efficiency
